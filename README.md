# YurtzCamp

YurtzCamp es un proyecto de practica pensado para enseñar a programar en equipo
de una forma simple, real y cercana. La idea principal es usar esta pagina como
un espacio de estudio donde dos personas puedan aprender juntas mientras
exploran herramientas fundamentales del desarrollo web y del trabajo
colaborativo.

Durante el proceso se pusieron en practica conceptos de HTML, CSS y JavaScript,
ademas del uso de Git y GitHub para versionar cambios, trabajar con repositorios
remotos y entender mejor el flujo de trabajo de un proyecto. Tambien se uso
Visual Studio Code con Live Share para compartir edicion, terminal y contexto en
tiempo real, lo que permitio aprender de manera mas dinamica y detectar errores
en conjunto.

YurtzCamp resume experiencias reales de aprendizaje: crear archivos, ordenar
carpetas, levantar un servidor local, usar Emmet para escribir mas rapido,
entender comandos como `git add`, `git commit`, `git push`, `git pull` y
`git remote -v`, y descubrir por que algunos errores aparecen cuando se trabaja
con cuentas distintas o permisos diferentes. Entre esos casos destaca el error
`403`, que deja en claro que compartir una terminal por Live Share no implica
compartir credenciales o permisos de GitHub.

Mas que una pagina estatica, YurtzCamp funciona como un registro de aprendizaje
compartido. Su objetivo es servir como base para enseñar a un compañero a
programar, mostrando no solo resultados, sino tambien dudas, errores,
correcciones y buenas practicas que forman parte del camino real de cualquier
desarrollador.
