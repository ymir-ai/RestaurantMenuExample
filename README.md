# YurtzCamp

YurtzCamp es un espacio de practica, documentacion y aprendizaje compartido. La idea del proyecto es usar una pagina real para enseñar programacion, registrar sesiones de pair programming, guardar errores comunes, explicar decisiones tecnicas y construir una ruta clara para crecer como desarrollador web moderno.

## Que hacemos

En este proyecto practicamos desarrollo web desde cero y en contexto real:

- escribimos HTML, CSS y JavaScript
- usamos Git y GitHub para versionar cambios
- trabajamos con Visual Studio Code y Live Share
- aprendemos a resolver errores de terminal, ramas, permisos y despliegues
- documentamos cada sesion para recordar que paso, que aprendimos y que mejorar

## Que queremos lograr

YurtzCamp busca convertirse en una base de estudio para formar a un desarrollador full stack capaz de diseñar, construir, desplegar y mantener productos reales. El objetivo no es solo aprender sintaxis, sino entender flujo de trabajo, arquitectura, buenas practicas, colaboracion, cloud, APIs, autenticacion, bases de datos, UX y despliegue continuo.

## Stack objetivo

El stack principal al que apunta YurtzCamp es MERN evolucionado con herramientas modernas:

- MongoDB para persistencia de datos
- Express y Node.js para backend
- React para interfaces
- Next.js para aplicaciones full stack y SSR
- Prisma como ORM y capa de acceso a datos
- Tailwind CSS y Material UI para velocidad y consistencia visual
- Redux Toolkit para manejo de estado
- Vercel para despliegue rapido
- AWS para almacenamiento, serverless y servicios complementarios

## Perfil profesional que inspira este camino

Este roadmap toma como referencia una experiencia real de trabajo remoto e independiente en desarrollo de software, full stack y producto digital. La meta es acercarse a un perfil capaz de:

- desarrollar frontend con HTML, CSS, JavaScript, TypeScript, React, Vue y Next.js
- construir backend con Node.js, Express, Flask o Django segun necesidad
- integrar ORMs y modelado de datos con Prisma, Mongoose y Sequelize
- trabajar con bases SQL y NoSQL
- desplegar en Vercel y servicios cloud
- usar Docker para entornos reproducibles
- integrar autenticacion con JWT, OAuth o servicios de terceros
- conectar pagos, correos, OCR y APIs externas
- documentar procesos, prototipar ideas y colaborar en equipo

## Vision de crecimiento

YurtzCamp no quiere ser solo una landing o una practica aislada. Quiere transformarse en un cuaderno vivo de aprendizaje tecnico, una guia de estudio full stack y una base para enseñar a otros a programar con ejemplos reales, errores reales y avances reales.
